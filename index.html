Starten von SolrCloud

Beispiel-Setup kann erzeugt werden mittels bin/solr -e cloud Anschließend werden alle erfolrderlichen Einstellungen interaktiv abgefragt.
Einen Knoten (solr node) herunterfahren mittels bin/solr stop -p 8984 Die Spezifikation des Knoten erfolgt über seinen Port.
Knoten wieder hochfahren mittels bin/solr start -cloud -p 8984 -s "example/cloud/node2/solr" -z localhost:9983 Als Wert des Parameters z wird die Adresse des Zookeeper-Servers angegeben.
Alle Knoten des SolrCloud Clusters herunterfahren mittels bin/solr stop -all

Starten von ZooInspector

ZooInspector ist ein GUI, um die im Zookeeper-Server abgelegten Konfigurationsdateien anzuzeigen. Änderungen werden von ZooInspector auch unterstützt.

Um ZooInspector zu starten muss in das Verzeichnis contrib/ZooInspector gewechselt werden und dort der Befehl java -cp ../../lib/*:lib/*:zookeeper-3.4.9-ZooInspector.jar:../../zookeeper-3.4.9.jar org.apache.zookeeper.inspector.ZooInspector ausgeführt werden.
